# HealthCare-SQL-Tableau
This project involves analyzing flu shot compliance and patient data for the year 2022. It combines SQL queries on patient and immunization data stored in a PostgreSQL database (using pgAdmin) with data visualization in Tableau.

## 2. Data Visualization in Tableau

* Total Compliance: Percentage of active patients who received flu shots in 2022.
* Flu Shots by Age: Bar chart showing compliance rates across age groups.
* Flu Shots by Race: Bar chart analyzing compliance among racial groups.
* Flu Shot % by County: Geographic distribution of flu shots using a map.
* Patient List: A table of patients with flu shot status.
* Yearly Flu Shot Trends: Line graph showing flu shot administration trends throughout the year.
